# 정부 정책 소개 상세페이지 — 표준 기획안

> 참조: 「상세페이지 표준 코너 구성안」의 5-ZONE / 13코너 프레임워크를
> 정부 정책 홍보에 맞게 재구성

---

## 설계 철학

상품 상세페이지의 핵심 원리 **"결핍 인식 → 해결책 → 혜택 증명 → 행동 유도"**는
정부 정책 소개에도 그대로 적용된다.

- "구매"를 "신청/참여"로 전환
- "제품 스펙"을 "정책 내용·지원 조건"으로 전환
- "리뷰"를 "수혜 사례·통계"로 전환
- "가격 혜택"을 "지원금·감면·혜택"으로 전환

## ZONE 1. 도입부 — 시선 고정 + 핵심 메시지

### 코너 1. 히어로 배너 / 정책 헤드라인

| 항목 | 내용 |
|------|------|
| **역할** | 3초 안에 "나와 관련 있는 정책"임을 인식시킨다 |
| **콘텐츠** | 정책명 + 한 줄 핵심 혜택 카피 + 대표 비주얼 |
| **카피 공식** | [대상자]를 위한 [핵심 혜택] — [정책명] |
| **예시** | "월세 부담에 지친 청년을 위한 월 최대 20만원 지원 — 청년 월세 특별지원" |

> 주의: 정책명만 단독 노출하면 이탈률이 높다. 반드시 **대상자 + 혜택**을 함께 보여준다.

---

### 코너 2. 핵심 혜택 요약 바

| 항목 | 내용 |
|------|------|
| **역할** | 스크롤 전에 핵심 정보를 한눈에 파악 |
| **콘텐츠** | 아이콘 + 짧은 텍스트 3~4개 나열 |
| **예시 항목** | 지원금액 / 신청기간 / 대상자 / 신청방법 |

```
[아이콘] 월 최대 20만원    [아이콘] 2026.3.1 ~ 12.31
[아이콘] 만 19~34세 청년   [아이콘] 온라인 5분 신청
```

### 코너 3. 신뢰 지표 / 정책 실적

| 항목 | 내용 |
|------|------|
| **역할** | "이미 많은 사람이 혜택을 받았다"는 대세감 |
| **콘텐츠** | 수혜자 수, 예산 규모, 관련 부처/기관 로고 |

## ZONE 3. 설득 — 정책이 어떻게 해결하는가

### 코너 6. 지원 내용 상세 (기능→혜택 변환)

| 항목 | 내용 |
|------|------|
| **역할** | 정책 스펙을 국민의 체감 혜택으로 번역 |
| **구성** | 체크포인트 1, 2, 3 형식으로 나열 |
| **변환 원칙** | 제도 용어 → 일상 언어 |

| 정책 스펙 (X) | 체감 혜택 (O) |
|---------------|--------------|
| 월 최대 20만원 지급 | 매달 월세 부담이 20만원 줄어듭니다 |
| 최대 12개월 지원 | 1년간 총 240만원까지 받을 수 있습니다 |
| 소득 기준 중위소득 60% 이하 | 연봉 약 2,600만원 이하면 신청 가능합니다 |

### 코너 8. 기존 제도 vs 신규 정책 비교표

| 항목 | 내용 |
|------|------|
| **역할** | 이전 제도 대비 개선점을 시각적으로 증명 |
| **구성** | 2~3열 비교 테이블 |

| 비교 항목 | 기존 제도 | 이번 정책 |
|-----------|-----------|-----------|
| 지원 금액 | 월 10만원 | **월 20만원** |
| 지원 기간 | 6개월 | **12개월** |
| 소득 기준 | 중위 50% | **중위 60%** |
| 신청 방식 | 방문 접수 | **온라인 5분** |

---

### 코너 9. 신청 절차 (스텝 바이 스텝)

| 항목 | 내용 |
|------|------|
| **역할** | "어렵지 않다"는 인식 심기 |
| **구성** | 3~5단계 프로세스 인포그래픽 |

```
STEP 1          STEP 2          STEP 3          STEP 4
자격 확인   →   서류 준비   →   온라인 신청  →   결과 확인
(1분)           (5분)           (5분)           (2주 내)
```

- 각 단계마다 필요 서류, 접속 링크, 소요 시간 명시
- "총 소요 시간: 약 10분" 강조

## ZONE 4. 확신 — 실제 효과와 사례

### 코너 10. 수혜 사례 / 국민 후기

| 항목 | 내용 |
|------|------|
| **역할** | 실제 혜택을 받은 사람의 목소리로 신뢰 확보 |
| **구성** | 인물 사진 + 이름(또는 이니셜) + 한 줄 후기 |

```
"월세 20만원을 지원받으니 적금을 시작할 수 있었어요"
— 김OO (28세, 서울 거주, 2025년 수혜자)

"신청이 이렇게 간단한 줄 몰랐어요. 10분이면 끝나요"
— 이OO (31세, 부산 거주)
```

- 가능하면 **숫자 데이터** 병행: 수혜자 만족도 94%, 재신청률 87% 등

## 레이아웃 원칙 (정책 버전)

| 원칙 | 적용 |
|------|------|
| **모바일 퍼스트** | 국민 대부분이 스마트폰으로 접근 — 세로 스크롤 최적화 |

| **접근성** | 고대비, 큰 글씨, 스크린리더 호환 |

## 상품 → 정책 전환 매핑표

| 상품 상세페이지 | 정책 상세페이지 |
|----------------|----------------|
| 제품명 | 정책명 |
| 가격/할인 | 지원금액/감면 혜택 |
| 제품 스펙 | 지원 조건·자격 요건 |
| 기능→혜택 변환 | 제도 용어→체감 혜택 변환 |
| 리뷰/후기 | 수혜 사례/국민 후기 |
| 구매하기 버튼 | 신청하기 버튼 |
| 패키지 구성 | 관련 정책/연계 혜택 |
| 배송/교환/반품 | 문의처/이의신청/FAQ |
| 비교표 (자사 vs 타사) | 비교표 (기존 제도 vs 신규 정책) |
| 환불 보장 | 재심사/이의신청 보장 |
